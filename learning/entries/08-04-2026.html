Learning Framer.

1. Anatomy of Framer Project.

The flow of a Framer project goes like this

  1. Workspace - Can have multiple projects
  2. Project - Each of which is essentially a website
  3. Pages - Each project can have multiple pages
  4. Design Canvas - Within each page we have design canvases
  5. BreakPoints - Different screen sizes for responsive design

Paging works like html where you can not rename home page, other pages you can rename like /about, which will lie in domain.com/about and which also can be made a folder,which will contain multiple other pages like/resumewhich will lie in domain.com/about/resume.
