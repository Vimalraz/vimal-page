Screenshot problem - a little tangent

I wanted to add an screenshot image for the empty C project screen in previous entry but screenshotting and saving it is a little difficult with windows default Snipping tool. Used AI to create a new screenshotting tool for my purposes.

  6. Activated by Ctrl + Shift + `
